# Httscrap

## Using

Code example:

  1. To log in fill login and pass in lib/httscrap aqnd run:

        Httscrap.start

  2. To download a file just run:

        Httscrap.download("http://example.com", "test.html")

  3. To get video run:

        Httscrap.get_video("http://example.com/video", "test.mp4")
